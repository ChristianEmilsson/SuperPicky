## 核心功能

![AI识别示意](img/comparison.jpg)

四大AI模型协同工作，多维度评估您的鸟类照片：

| 序号 | 功能模块 | 作用 |
|:---:|---------|------|
| 01 | **YOLO11 检测** | 精准识别鸟类位置 |
| 02 | **SuperEyes 鸟眼** | 检测眼睛可见度 |
| 03 | **SuperFlier 飞鸟** | 识别飞行姿态加分 |
| 04 | **TOPIQ 美学** | 评估构图与美感 |

---

## 评分规则

根据**头部锐度**和**美学评分**自动评定星级：

| 星级 | 评判标准 | 建议 |
|:---:|----------|------|
| ★★★ | 锐度+美学双达标 | **精品** 值得后期精修 |
| ★★☆ | 锐度或美学达标其一 | 良好，可考虑保留 |
| ★☆☆ | 有鸟但未达标 | 普通，可以删除 |
| 0 | 无鸟 / 质量太差 | 放弃，建议删除 |

## 特殊标记

### Pick 精选标记

从三星照片中选出锐度和美学**双排名前25%**的交集——这些就是这批照片的精华！

### 飞鸟绿标

AI识别到飞行姿态的照片用绿色标签标记，并获得额外加分。飞版难拍，理应获得更多认可！

### 曝光检测（可选）

可开启曝光问题检测，过曝或欠曝的照片会自动降星。默认关闭，需手动开启。

---

## 自动整理

处理完成后，照片自动按星级分类到不同文件夹：

```
📁 您的照片文件夹/
   ├── 3_stars/  ← 三星精品
   ├── 2_stars/  ← 二星良品
   ├── 1_stars/  ← 一星普通
   └── 0_stars/  ← 零星淘汰
```

## 使用方法

### 第一步：选择文件夹

点击「浏览」按钮，选择包含照片的目录

![选择目录](img/tutorial/step2_select.jpg)

### 第二步：调整参数

推荐设置：锐度 400，美学 5.0，勾选飞行姿态

![参数设置](img/tutorial/step3_params.jpg)

### 第三步：开始处理

点击按钮，等待AI完成分析（约1秒/张）

![处理中](img/tutorial/step4_processing.jpg)

### 第四步：查看结果

照片自动按星级分类，显示详细评分信息

![选片结果](img/tutorial/step5_result.jpg)

## 为什么眼睛重要

**眼睛是鸟类摄影的灵魂！**

| 眼睛状态 | 评分影响 |
|---------|---------|
| 眼睛清晰可见 | → 正常评分 |
| 眼睛略微模糊 | → 最高2星 |
| 眼睛完全不可见 | → 最高1星 |
